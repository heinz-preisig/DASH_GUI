📖 Form Help

Quick Start

  1. Click "Generate Form" to create input fields
  2. Fill in your data (required fields marked with *)
  3. Click "Save RDF Data" to generate RDF

Field Types

  • Text Fields: Any string input
  • Date Fields: Use date picker (format: YYYY-MM-DD)
  • Email Fields: Email format (user@example.org)

Tips

  • Save frequently - RDF only generates when you click "Save RDF Data"
  • Check the RDF Output panel to verify your data
  • Required fields must be filled for valid RDF

Troubleshooting

  • Form not generating? Check browser console for errors
  • RDF not saving? Verify all required fields are filled
  • Invalid output? Check field formats (dates, emails)
